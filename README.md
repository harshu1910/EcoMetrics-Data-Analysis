\# EcoMetrics Data Analysis



EcoMetrics is a data analysis project focused on environmental sensor data such as temperature, humidity, CO2 levels, and air quality.  

The project explores patterns and trends in environmental conditions using data analytics techniques.  

Python and Power BI are used to process, analyze, and visualize the dataset.  

The goal is to understand environmental changes and build insights useful for smart monitoring systems.
